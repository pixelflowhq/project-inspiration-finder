## 🛠 Dev Tools

**Wuphf — AI-Maintained Wiki (Show HN)**
AI agents auto-create and maintain a knowledge base as Markdown files in Git — Karpathy-style. Brilliantly simple architecture: Git gives you versioning and diffs for free, and Markdown keeps everything portable. Buildable in a weekend or two.
→ [github.com/nex-crm/wuphf](https://github.com/nex-crm/wuphf)

**Euphony**
Turns raw AI chat data and Codex logs into clean, browsable, human-readable views. If you've ever scrolled through a 200-turn agent log trying to find where things went wrong, you know why this matters. Quick build potential.

**Architecto**
AI-powered design, review, and documentation of cloud architecture diagrams. Cloud diagrams rot faster than code comments — an AI that keeps them honest is genuinely useful infra tooling.
→ [producthunt.com](https://www.producthunt.com/products/architecto?utm_campaign=producthunt-api&utm_medium=api-v2&utm_source=Application%3A+product-inspiration-finder+%28ID%3A+282730%29)

**Kloak (Show HN)**
