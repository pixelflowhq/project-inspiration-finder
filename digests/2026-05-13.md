## AI/ML

**Needle: Distilled Gemini Tool Calling into a 26M Model**
A 26-million-parameter model distilled from Gemini that handles tool/function calling for AI agents. If you need structured function invocation without the cost and latency of a full LLM, this is a serious contender for embedded agent architectures.
→ [github.com/cactus-compute/needle](https://github.com/cactus-compute/needle)

**Statewright – Visual State Machines for Reliable AI Agents**
A framework for building AI agents as visual state machines instead of ad-hoc prompt chains. Makes agent behavior predictable, inspectable, and debuggable — the kind of guardrails production agents desperately need.
→ [github.com/statewright/statewright](https://github.com/statewright/statewright)

**MiniCPM-V 4.6**
A 1.3B parameter vision-language model designed to actually run on mobile devices. Multimodal AI on-device without cloud round-trips opens up offline assistants, camera-based tools, and accessibility apps.

**Free AI SEO Auditor**

→ [producthunt.com/products/free-ai-seo-auditor](https://www.producthunt.com/products/free-ai-seo-auditor)

**ContentPilots**
