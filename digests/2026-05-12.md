Companion piece to Known Agents. Simple, free, viral potential—the kind of tool that gets shared in every ops Slack channel.

**safe-install — Trusted NPM Build Dependencies**
An npm wrapper that validates install scripts against a trusted allowlist to block supply chain attacks.
Supply chain security is an evergreen problem. This is a weekend build that solves a real, recurring fear.

**OpenGravity — Zero-Install, BYOK Antigravity Clone**

**CloakBrowser — Stealth Chromium with Source-Level Anti-Detection**

The arms race between scrapers and bot detection is intensifying. This is a power tool for the automation community.

**SuperSplat — Browser-Based 3D Gaussian Splat Editor**
